# **App Name**: Heartfelt Proposal

## Core Features:

- Message Display: Display romantic messages sequentially with animated transitions.
- Heart Burst: Trigger a burst of hearts upon clicking 'Next'.
- Image Integration: Display a mock image from https://picsum.photos/seed/love/400/500 leading to the question.
- 'Maybe' Button Dodge: Implement a 'Maybe' button that dodges user interaction on mobile.
- Celebration Animation: Display a full-screen success celebration with confetti animation on 'Yes'.

## Style Guidelines:

- Primary color: Soft pink (#F4C2C2) to evoke feelings of love and warmth.
- Background color: Light lavender (#E6D7ED) to provide a soft, dreamy backdrop, enhancing the romantic atmosphere. It has the same hue as pink, but is desaturated and lightened for a gentle background.
- Accent color: Lilac (#B6A5C8) to create visual interest and complement the primary pink hue.
- Font: 'Alegreya', a humanist serif with an elegant, intellectual, contemporary feel; suitable for headlines or body text
- Mobile-first, responsive design with a central glassmorphism card for content.
